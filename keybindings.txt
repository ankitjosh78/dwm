Most Used Keybindings:

Win + (Number[1-9])-->respective tag

Win + Enter -->st (terminal)

Alt + Ctrl + t --> alacritty (terminal)

Win + Shift + Enter --> nautilus (file manager)

Win + Shift + b --> brave (browser)

Win + p --> dmenu (run menu)

Win + q --> kill application window

Win + Shift + d -->discord

Win + <F12> --> rofi launcher (run menu)

Win + o --> opens scratchpad (default st)

Win + Shift + o --> obs

Win + Shift + n--> nitrogen

Win + Shift + r-->Quit Dwm(or restart if you have an infinite loop in you .xinitrc)

* Rest can be found by looking in the config.h file *
